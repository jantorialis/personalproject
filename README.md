# personal website

a personal website for myself, featuring a retro aesthetic with modern interactive elements.

![website preview](https://files.catbox.moe/t7kj4y.gif)

## features

- **interactive 3d weathervane** - spinning model on the entrance page built with three.js
- **music player** - listen to my favorite tracks with my custom playlist
- **art gallery** - clickable thumbnails with lightbox navigation
- **responsive layout** - works on desktop, tablet, and mobile
- **journal section** - a log for my thoughts

## pages
---

| page | description |
|------|-------------|
| `index.html` | entrance page with 3d spinning weathervane |
| `main.html` | home page with navigation to other parts of the site |
| `about.html` | about the creator ( me ) |
| `portfolio.html` | art gallery with lightbox viewer |
| `music.html` | custom music player with playlist |
| `journal.html` | personal journal entries |
| `sitemap.html` | site navigation map |

## technologies used

- **html5** - structure and content
- **css3** - custom styling
- **javascript** - interactivity and functionality
- **three.js** - 3d weathervane on entrance page
